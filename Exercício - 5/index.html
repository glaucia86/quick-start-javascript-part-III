Programação Orientada a Objetos

JavaScript usa objetos como uma maneira de passar e compartilhar códigos e funcionalidades. Quando criamos variáveis, funções, e objetos, são propriedades de objeto global

O window é um objeto global quando usamos um host da web. Quando escrevemos uma função que não está associada com um objeto, se torna um método do objeto global. Por exemplo, se escrevemos uma função chamada 'olaMundo()', podemos também chamar o método 'window.olaMundo()'.

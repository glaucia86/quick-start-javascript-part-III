Entendendo os Objetos

Nossos amigos estão de volta! Mas... vamos trabalhar com objetos com eles

Quando nós criamos nosso objeto 'myCoffee', criamos usando de uma maneira que chamamos de objeto literal. Quando criamos objetos dessa maneira, nós estamos definindo e criando um único objeto, a qual referenciamos como instância de um objeto. que nós usamos uma função para criar múltiplos instancias de um objeto. Cada instância heranças das propriedades e métodos de um determinado construtor.

Veja a sintaxe: Objetos Literais

                    var myPrimeiroObjetoLiteral = {               
                  //adicionar propriedades necessárias:
                  exemploChave: exemploValor
                
                };               
               function MyPrimeiroConstrutor(exemplo) {               
                  //adicionar propriedades necessárias:
                  this.exemplo = exemplo;               
                };

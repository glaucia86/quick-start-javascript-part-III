Objetos

Um objeto é uma coleção de propriedades, que são descritas na forma de key / values. Podemos usar os objetos para modelar coisas, usando código.

Veja a sintaxe: Objetos

                var meuPrimeiroObjeto = {               
                  //adicionar propriedades necessárias:
                  chaveExemplo: valorExenplo              
                };

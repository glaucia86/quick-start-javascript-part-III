Entendendo Escopo

Ao compreender o escopo do código, nós sabemos que afetará somente uma parte do nosso código ou todo o código base. Se criarmos coisas que são globais no escopo, estaremos dando o poder para controlar o nosso código. Com isso, devemos ser muito cuidadosos e procurar cuidar do nosso código quando criamos coisas que estão sendo criadas com um escopo global. Especialmente importante se você planeja usar bibliotecas de JavaScript, como por exemplo: JQuery.

Escopo Local

JavaScript usa function scope ou seja, todas as vezes que nós criamos uma nova função o escopo mudará. Qualquer código dentro da função é local para aquela determinada função. Nesse caso, esse código está no escopo local e não é acessível para ser usado fora do código.

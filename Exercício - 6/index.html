Escopos Globais e Locais

Na última lição aprendemos que quando escrevemos um código no browser se torna parte de um window object global. A maioria dos códigosque temos escrito até aqui tem sido de maneira global. Quando algo global significa que todos os códigos podem ter acesso, mudança. Isso significa que nós temos que ser bem cuidadosos sobre criar qualquer coisa que seja de escopo global. Senão acidentalmente podemos fazer algo que nós não tínhamos intenção.

Quando criamos uma variável dentro de uma função, não podemos acessar fora do código da função. Por que? Porque é uma variável de escopo local. na função. Isso significa que ela estará invisível para qualquer outro código que outro código dentro daquela função.
